📦 PROJECT OVERVIEW
═══════════════════════════════════════════════════════════════════════════

A complete tablet feedback collection system with real-time event management
and analytics. Includes:

  ✅ Tablet UI (/)
     • Full-screen feedback interface
     • 5-level emotion buttons (Terrible → Great)
     • Real-time event detection (2-second polling)
     • "Thank You" confirmation message

  ✅ Admin Dashboard (/admin)
     • Password-protected login
     • Event creation & management
     • Real-time event activation
     • Live analytics dashboard with charts

  ✅ Public Analytics (/analytics)
     • Real-time statistics
     • No authentication required
     • Auto-refreshing data (5 seconds)
     • Perfect for public displays

  ✅ Backend System
     • PostgreSQL database with indexes
     • 5 API endpoints
     • JWT authentication
     • Real-time polling mechanism

🚀 GETTING STARTED (3 STEPS)
═══════════════════════════════════════════════════════════════════════════

STEP 1: Install Dependencies
  $ npm install

STEP 2: Setup Database
  $ cp .env.example .env.local
  # Edit .env.local with your PostgreSQL URL
  $ npm run db:migrate

STEP 3: Run Development Server
  $ npm run dev

  Then visit:
  • Tablet UI:      http://localhost:3000
  • Admin:          http://localhost:3000/admin     (password: admin123)
  • Analytics:      http://localhost:3000/analytics
