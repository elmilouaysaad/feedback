## 🚀 Ready to Use

### Step 1: Install
```bash
npm install
```

### Step 2: Configure Database
```bash
cp .env.example .env.local
# Edit DATABASE_URL with your PostgreSQL connection
```

### Step 3: Initialize Database
```bash
npm run db:migrate
```

### Step 4: Run Development
```bash
npm run dev
```

### Step 5: Access System
- **Tablet UI**: http://localhost:3000
- **Admin Dashboard**: http://localhost:3000/admin (password: `admin123`)
- **Public Analytics**: http://localhost:3000/analytics
